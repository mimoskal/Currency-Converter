# Currency Converter
Sample React app.

### Getting Started
- Install dependencies

	`yarn install`
- Run development server (localhost:8888)

	`yarn start`
- Run mocha tests

	`yarn test`
- Run ESLint

	`yarn lint`
- Build production package

	`yarn build`
